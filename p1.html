Design Inspiration

  • Neal.fun Some cool webpages here! The password game is one that got really popular recenly

  • What Football Will Look Like In The Future My favorite online/interactive story ever! Does some really neat things with multimedia, and the first time you click the link is always a fun experience :-)

  • Escape Velocity Webcomic that a friend of a friend made, does some stuff I like with storytelling as well as adding music to the pages

  • Letterboxd I use this site a lot.... I really like the way it's laid out and consider it to be a much better design than goodreads, which I also use

  • Last.fm Music site that records whatever you listen to-- it's really useful if I hear a song I like but forget what it was called lol

  • Honorable mention is Blaseball-- really fun website while it existed and did some fun interactive stuff I'd like to learn to incorperate but it isn't running anymore :(
